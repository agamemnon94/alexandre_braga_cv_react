# Création de projet REACT VITE

```
npm create vite@latest
```

### - Puis nom du projet

### - Puis select framework avec les ↑↓

### - cd "dossier du projet"

### Puis installer les dépendances

```
npm install
```

```
npm i react-router-dom
```

### - Puis :

```
npm run dev
```
